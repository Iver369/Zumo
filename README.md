```
git clone https://github.com/Iver369/Zumo.git (laster ned hele prosjektet fra github til vscode)
git status (viser hvilke filer som er endret)
git pull (oppdaterer prosjektet med siste endringer, bruk om du ikke har gjort endringer)
git add . (legger til alle endrede filer for commit)
git commit -m "Hva du har gjort" (lagrer endringene lokalt med beskrivelse)
git pull --rebase (oppdaterer prosjektet med siste endringer og legger dine oppå de nyeste, bruk om du har gjort endringer)
git fetch (ser siste endringer, men legger ikke til endringene, bruk dersom du bare vil se endringer uten å legge dem inn)
git stash (midlertidig lagrer endringer før du oppdaterer, bruk dersom du har filer som ikke er lagret og som andre har arbeidet på)
git stash pop (henter tilbake endringer som ble lagret med git stash, etter at prosjektet er oppdatert)
git push (laster opp endringene til github)

Extensions: PlatformIO, C/C++
```
